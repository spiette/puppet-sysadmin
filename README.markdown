# sysadmin
[![Build Status](https://travis-ci.org/spiette/puppet-sysadmin.png)](https://travis-ci.org/spiette/puppet-sysadmin)

This is the sysadmin module. It installs packages for system administration.

* acpid
* blktrace
* curl
* dstat
* ethtool
* iotop
* iftop
* iptraf(-ng)
* iperf
* mtr
* screen
* sysstat
* traceroute
* wget
